Creacion de arboles binarios

   Si se desea agregar datos ingrese dentro del campo de Datos de entrada, debe separar los numeros por una coma por ejemplo: 1,2,3,4,5,6.

  A continuacion debe seleccionar el boton de agregar y a continuacion se generara el arbol binario.

  

   Una vez generado el arbol puede mostrar el orden con el boton ¨Motrar orden¨ y se le mostrara en forma de alerta el in-order, pre-order, post-Orden

   Tambien puede generar el arbol a partir de una secuencia de orden, para esto debe inresar la secuencia ya ordenada en los espacios ¨Ingrese el pre order¨ e ¨Ingrese el post order¨ respectivamente y presionar el boton ¨Generar arbol¨, debe asegurarse de que la secuencia es correcta para que se pueda generar el arbol.
